JARGON ALGORITHM 66: Password Multiple Attempts

SET password ("open")
SET correct ("False")

REPEAT 3 times
    ASK "Enter password" as attempt
    IF attempt is equal to password THEN
        PRINT "Access granted"
        SET correct ("True")
        BREAK
    ELSE
        PRINT "Wrong password"
    END
END

IF correct is not equal to "True" THEN
    PRINT "Access denied"
END

Expected Output (for wrong input 3x):
Wrong password
Wrong password
Wrong password
Access denied

Explanation:
- Loops up to 3 times for password input.
- Uses BREAK if password is matched.
