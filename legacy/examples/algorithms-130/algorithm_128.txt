Algorithm 128: Reverse List Manually
SET original ([1, 2, 3, 4])
SET reversed ([])
SET i (0)

REPEAT_UNTIL i reaches end of original
    SET index ((4 - 1) - i)
    ADD original[index] to reversed
    SET i (i + 1)
END

PRINT reversed

Expected Output:
[4, 3, 2, 1]

Explanation:
Reverses a list using manual indexing. No built-in reverse function.
