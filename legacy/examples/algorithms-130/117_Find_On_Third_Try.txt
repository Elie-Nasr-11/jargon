Title: Find on Third Try
Jargon Code:
SET tries (0)
SET found ("False")

REPEAT_UNTIL found is equal to "True"
    SET tries (tries + 1)
    IF tries is equal to 3 THEN
        PRINT "Found on third try!"
        SET found ("True")
    ELSE
        PRINT "Not yet..."
    END
END

Expected Output:
Not yet...
Not yet...
Found on third try!

Explanation:
Looping with a flag until a certain condition is met, simulating a search with feedback.
