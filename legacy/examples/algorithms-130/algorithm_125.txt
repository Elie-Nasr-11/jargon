Algorithm 125: Count Occurrences
SET items ([1, 3, 3, 7, 3, 2])
SET target (3)
SET count (0)
SET i (0)

REPEAT_UNTIL i reaches end of items
    IF items[i] is equal to target THEN
        SET count (count + 1)
    END
    SET i (i + 1)
END

PRINT count

Expected Output:
3

Explanation:
Counts how many times a value appears in a list.
